# 垚（yao）芽熹｜完整版上线版

## 你要上传到 GitHub 的文件
- index.html
- qr.png

## 最傻瓜部署
1. 打开 GitHub 仓库
2. Add file -> Upload files
3. 把整个文件夹里的 `index.html` 和 `qr.png` 一起拖进去
4. Commit changes
5. 打开 Vercel，用 GitHub 登录
6. New Project -> 选择这个仓库 -> Deploy

## 后期修改
- 改文案：直接编辑 `index.html`
- 换二维码：把 `qr.png` 替换成新图，文件名保持不变
